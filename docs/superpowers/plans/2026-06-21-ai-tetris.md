# AI 대전 테트리스 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.
>
> **참고(환경):** 이 환경에는 `git`이 없습니다. 각 태스크 끝의 "Checkpoint" 단계는 상태 저장 지점을 의미하며, git이 없으므로 실제 커밋 명령은 생략합니다(나중에 git 도입 시 `git add`/`git commit`으로 대체).

**Goal:** 플레이어 vs AI(난이도 1·2·3) 공격형(가비지) 테트리스를 네온 신스웨이브 3D로 구현한 로컬 실행 웹앱을 만든다.

**Architecture:** 코어 게임 로직(tetromino/board/randomizer/rules/engine/ai)은 프레임워크 무관 순수 TS로 단위 테스트 가능하게 분리. `versus/match`가 두 엔진을 연결해 가비지를 라우팅. `render`(Three.js)가 두 필드를 3D로 그리고 연출을 담당. `ui`(DOM)가 HUD·메뉴 오버레이. `main`이 고정타임스텝 로직 + RAF 렌더 게임 루프를 구동.

**Tech Stack:** TypeScript(strict), Vite, three.js(+ examples/jsm postprocessing: EffectComposer/UnrealBloomPass), Vitest. 오디오는 WebAudio 직접 합성(에셋 없음). 의존성: three, vitest, typescript, vite. (howler 등 외부 오디오/음원은 사용하지 않음.)

**Spec:** `docs/superpowers/specs/2026-06-21-ai-tetris-design.md`

## Task 3: board.ts (그리드/충돌/라인클리어)

**Files:** Create `src/core/board.ts`; Test `tests/core/board.test.ts`

- [ ] **Step 1: 실패 테스트**
```ts
import { Board } from '../../src/core/board';
import { getShape } from '../../src/core/tetromino';

describe('board', () => {
  it('빈 보드에서 바닥 충돌 없이 이동 가능', () => {
    const b = new Board();
    expect(b.collides(getShape('I',0), 3, 0)).toBe(false); // 상단
  });
  it('바닥/벽/쌓인 블록 충돌 감지', () => {
    const b = new Board();
    expect(b.collides(getShape('I',0), 3, 21)).toBe(true);  // 바닥 초과
    expect(b.collides(getShape('I',0), -1, 0)).toBe(true);  // 좌벽
  });
  it('조각을 고정하면 셀이 채워진다', () => {
    const b = new Board();
    b.place(getShape('O',0), 4, 20, 'O');
    expect(b.getCell(4,20)?.kind).toBe('O');
  });
  it('완성된 행을 클리어하고 개수를 반환', () => {
    const b = new Board();
    for (let x=0;x<10;x++) b.setCell(x,20,'X');
    expect(b.setCell(0,20,'X')); // ensure full
    const cleared = b.clearLines();
    expect(cleared).toBe(1);
  });
});
```
